Hello! Active-status

Adding class name in a few seconds by event action, Maybe milli seconds. Then you can add css3 animation in the same time. This plugin depends on jQuery, link the jquery at first.

Active-status Example

Active-status Usage

Default opition

$('.testdom').astatus()

Add function

$('.testdom').astatus(function(){
  //...do something
})

Effect on another element (All opitions)

$('.testdom').astatus({
  delay:300, //Delay time
  etype:'click', //Event type, click or touchstart
  cname:'active-status', //Class name
  adddom:'.test-anotherdom' //Other element toggle class name
},function(){
  //...do something
})

CSS Animation

.testdom{
  -webkit-transition: all 300ms;
  -o-transition: all 300ms;
  transition: all 300ms;
}
.testdom.active-status{
  -webkit-transition: all 300ms;
  -o-transition: all 300ms;
  transition: all 300ms;
}
